# 🏡 3D 治愈群岛 & 虚拟小屋 (3D Home)

欢迎来到 **3D 治愈群岛与虚拟小屋** 个人主页系统！这是一个基于 **Three.js** 与原生 **HTML5/CSS3** 构建的、极具现代设计美学与丰富动态交互的 3D 个人主页。它打破了传统扁平网页的限制，让访客可以通过控制可爱的 3D 角色，在一个充满氛围感的悬浮岛屿与双层海滨小屋中进行探索。

---

## 🌟 核心特色与功能

### 1. 双层场景切换与动态加载
* **海滨浮空岛 (Island Map)**：阳光沙滩、椰子树、海浪波纹、可抱起并投掷的物理沙滩球、交互式秋千。
* **复古双层小屋 (Cottage Map)**：通过岛上的温馨木屋直接传送至宽敞的小屋内部。小屋内包含双人床、绿植地毯、换装衣柜、调色画架及预留的扩展家具槽位。

### 2. 交互式写生画架 (Wet-Blending Easel)
* **独立写生页面**：采用独立沙箱页面 `paint.html`，实现移动端横屏响应式适配。
* **上下铺满与无缝边框**：移动端横屏下画布自动延伸并最大化垂直铺满屏幕（`100vh`），去除 top/bottom 边框与圆角，提供最大化作画视野。
* **圆形 SVG 返回图标**：废弃了冗长的文字按钮，在左侧工具栏顶部新增精致的圆形 SVG 返回箭头图标，具备高颜值的悬停微动效，完美释放垂直高度。
* **物理湿混逻辑**：模拟水彩/油画的物理混色，画笔在划过画布已有色块时，会以 75% 笔触色 + 25% 画布底色进行物理湿混融合。
* **调色板融合与稀释**：支持调色板“融合”（50%/50% 混色）与“加水稀释”（调和纯白）操作，具备 15 步历史撤销功能。
* **色彩还原度比对**：通过计算用户绘制区域与随机写生目标色的 RGB 欧氏距离空间 Delta E 平均值，实时反馈匹配相似度百分比。

### 3. 三维实时换装系统 (3D Avatar Customizer)
* **多模型切换**：支持选择“海滨小女孩”、“海滨小男孩”以及“金铃铛萌猫”等多套 3D 基础网格。
* **透明雾化交互视窗**：换装 UI 适配移动端横屏，中央悬浮展示 3D 角色模型，两侧以玻璃摩砂卡片形式呈现角色模型切换参数控制。
* **字号与图标精细调优**：将弹窗及侧边栏标题（如“选择角色”、“定制色彩”）与对应的头部 Emoji 图标大小精细化缩减（如移动端标题降至 `1.05rem`），并全局拦截连击产生的文字选中和词典翻译弹窗，呈现更加利落高级的信息层级。

### 4. 氛围感夜间霓虹与彩灯串 (Vibrant Lighting)
* **霓虹彩灯街灯**：岛上街灯换装为粉红、青蓝、荧光绿的霓虹霓彩灯。
* **悬挂彩灯串 (Fairy Lights)**：使用 `THREE.CatmullRomCurve3` 与 `THREE.TubeGeometry` 在小屋屋顶前檐、秋千横梁、街灯之间挂载多条微重力下垂的彩色发光灯串，夜间自动泛出呼吸彩光。
* **昼夜交替系统**：可在小屋床前进行躺下交互，触发昼夜交替。夕阳与暗夜时分，室内吊灯、台灯、室外篝火、火把及彩灯串将自动亮起并伴随篝火微弱闪烁效果。

### 5. 极佳的移动端适配与双重感应
* **双通道手势触控**：左侧屏幕动态定位虚拟摇杆，右侧屏幕支持单指滑屏旋转 3D 视角。
* **边缘防误触安全区**：在屏幕最左侧与最右侧设立 55px 的触摸安全屏蔽区（限制摇杆生成范围），彻底避免操作时触发移动端浏览器原生的“侧滑返回/前进”或隐藏工具栏的手势冲突。
* **自动全屏模式**：在移动端检测到玩家的第一次触屏交互（touch/click）后，程序会自动且合法地切入 fullscreen 全屏游戏状态，完美隐藏导航栏与地址栏。
* **双重端感应防护**：通过 `@media` 查询与 JS `touchstart` 触控点感应双重机制，自动在移动设备/模拟器上应用 `is-mobile` 状态，强制启用右侧操作按钮（加速 🏃、跳跃 🦘、交互 ✨），彻底避免按钮缺失问题。
* **方向锁屏提示**：在手机竖屏状态下自动弹出动效遮罩，提示访客旋转手机横屏体验。

### 6. 自研复古电子乐合成器 (Synthesized Chimes)
* **纯代码无音频依赖**：基于 Web Audio Context API 实时编程合成，无任何 MP3 静态资源请求。
* **动态和弦音步**：根据昼夜状态切换主基调，模拟清脆温润的八音盒/冬日风铃的五声音阶琶音，伴随篝火踢球声反馈。

## 🛠️ 技术栈与架构

* **三维引擎**：[Three.js (r160)](https://threejs.org/)
* **构建工具**：[Vite 5](https://vitejs.dev/)
* **样式系统**：CSS3 变量、毛玻璃（Glassmorphism）、微交互动画
* **逻辑控制**：原生 ES Modules 规范 JavaScript

### 📂 目录结构

```
3d-home/
├── dist/                # 编译打包输出目录
├── public/
│   └── paint.html       # 独立写生画画页面 (Vite 静态目录，构建后输出至根目录)
├── src/
│   ├── ui/
│   │   └── Modal.js     # UI 弹窗、大厅 iframe 加载与 site-config.js 动态数据绑定
│   └── world/
│       ├── BeachBall.js  # 沙滩球/雪球物理、碰撞反弹与抱起抱跑逻辑
│       ├── Environment.js # 天空盒、雾化、太阳光及昼夜平滑过渡控制
│       ├── House.js     # 小屋 3D 几何、家具摆设、室内光源及浮空风景渲染
│       ├── Interacts.js # 交互感应检测、捡起/扔球、传送点及 Modal 事件派发
│       ├── Island.js    # 地形、植被、摆设、霓虹路灯、悬挂发光彩灯串
│       └── Player.js    # 角色碰撞体检测、重力/物理运动、摄像机轨道插值、换装形变
├── index.html           # 3D 页面主入口，包含 HUD 布局、换装、各种 Modal 结构
├── main.js              # 全局 GameApp 初始化、Vite 调试、移动端双触控输入及合成器
├── style.css            # 核心样式系统，毛玻璃卡片设计，全尺寸响应式布局
└── vite.config.js       # Vite 服务器、动态 parent middleware 配置及打包编译参数
```

## 🎮 控制指南

### 💻 电脑桌面端 (PC)
* **移动**：`W` `A` `S` `D` 或 `↑` `↓` `←` `→`
* **视角旋转**：鼠标拖拽空白区域
* **疾跑**：按住 `Shift` 键
* **跳跃 / 站起**：`Space` 空格键
* **功能交互**：靠近物体时按 `E` 键（包含传送、躺下、换装、写生、查看项目等）

### 📱 移动手持端 (Mobile)
* **移动**：触摸左侧屏幕任意区域动态生成 **虚拟摇杆** 拖拽移动
* **视角旋转**：单指拖拽右侧非按钮区域
* **加速**：按住右下角 🏃 按钮
* **跳跃 / 下秋千 / 站起**：点击右下角 🦘 按钮
* **功能交互**：靠近交互点时，右侧会自动亮起 ✨ 按钮，点击即可触发交互

## 🚀 本地开发与部署

请确保本地已安装 [Node.js](https://nodejs.org/) 环境。

### 1. 安装依赖
在项目根目录 `d:\code\secondhand.com\3d-home` 执行：
```bash
npm install
```

### 2. 运行本地开发服务器
启动开发服务器进行热调试：
```bash
npm run dev
```
启动后可在浏览器访问控制台输出的地址（默认为 `http://localhost:3000`）。

### 3. 项目打包编译
生成用于生产环境部署的压缩静态资源：
```bash
npm run build
```
编译完成后，所有的静态 HTML, CSS 和 JS 将输出至 `dist/` 文件夹，可以直接部署至任何静态网页服务器。
